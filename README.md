Este es un **proyecto de ejemplo de cliente SOAP con Spring Boot** que demuestra cómo consumir servicios web SOAP desde una aplicación Java. El proyecto utiliza Spring Web Services (Spring-WS) con JAXB para el marshalling/unmarshalling de XML, e implementa seguridad WS-Security con autenticación basada en usuario y token (credenciales hardcodeadas: admin/secret). La aplicación se conecta a un servicio web de gestión de tiendas en `http://localhost:8080/ws` y consulta información de tiendas incluyendo detalles del gerente, utilizando clases generadas desde un WSDL que definen el modelo de datos (Store, Manager, Role). Al ejecutarse, el programa realiza una petición de prueba para obtener la tienda con ID 3 usando dos métodos diferentes del cliente SOAP e imprime el resultado en consola, sirviendo como ejemplo educativo de integración con servicios SOAP en Spring Boot 2.7.3 con Java 11.
